# Axé Fadakà – Agenda do Babalorixá

Estrutura inicial em Flutter usando Clean Architecture e MVVM (Riverpod). Projeto preparado para Firebase (Auth/Firestore) mas com datasource mock por enquanto.

Rodar:

1. Instalar dependências: `flutter pub get`
2. Rodar no emulador/dispositivo: `flutter run`

Testes e análise:

- Executar análise estática:

```bash
flutter analyze
```

- Executar testes unitários:

```bash
flutter test
```

Observação: este ambiente de edição não possui o SDK do Flutter configurado; execute os comandos localmente.
